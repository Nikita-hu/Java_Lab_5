**Проект**: Java_Lab_5

- **Краткое описание**: набор учебных задач на Java: работа с дробями, вычисление симметрической разности списков, обработка результатов тестирования абитуриентов, поиск звонких согласных в тексте и демонстрация разворота очереди через стек.

**Файлы проекта**
- `Main.java`: главный класс с текстовым меню. Запускает подсистемы по выбору темы:
  - `1.1` — работа с дробями (`Fraction`, `FractionDemo`)
  - `3.9` — формирование списка L (симметрическая разность) (`ListSymmetricDifference.java` → класс `SymmetricDifferenceProcessor`)
  - `4.4` — обработка результатов тестирования (`TestingResultsProcessor`, `Applicant`, `TestingDemo`)
  - `5.5` — поиск звонких согласных в тексте (`VoicedConsonantFinder.java` → класс `VoicedConsonantAnalyzer`)
  - `6.1` — демонстрация разворота очереди через стек (`ReverseQueueArrayStack.QueueReverser`)

- `Fraction.java`: содержит `Fraction`, реализующий интерфейс `FractionOperations`. Поддерживает нормализацию знака, проверку знаменателя, кэширование вещественного значения, сеттеры/геттеры и метод сокращения.

- `FractionDemo.java`: демонстрация использования `Fraction` (создание, кэширование, сравнение, сокращение, ввод).

- `ListSymmetricDifference.java`: файл с реализацией симметрической разности — класс `SymmetricDifferenceProcessor`.

- `ReverseQueueArrayStack.java`: содержит вложенный статический класс `QueueReverser`, который принимает `Queue<Integer>` и печатает элементы в обратном порядке.

- `Applicant.java`: модель абитуриента с валидацией баллов, геттерами и форматированным выводом.

- `TestingResultsProcessor.java`: парсинг строк с данными абитуриентов, хранение списков и подсчёт статистики.

- `TestingDemo.java`: демонстрация работы обработки тестовых данных.

- `VoicedConsonantFinder.java`: содержит `VoicedConsonantAnalyzer` и `RussianTextUtils` для поиска звонких согласных в тексте; реализована загрузка из файла.

- `file_task5.txt`: пример входного текста для задачи `5.5` (поместите в корень проекта).

**Примечания и рекомендации**
- В файле `ListSymmetricDifference.java` основной класс называется `SymmetricDifferenceProcessor` — это указано выше для ясности.
- Файл `VoicedConsonantFinder.java` содержит основной класс `VoicedConsonantAnalyzer` (имена файла и класса различаются — это нормально, но указано явно).
- Для корректной работы задачи `5.5` `file_task5.txt` рекомендуется хранить в кодировке UTF-8. Если входной файл в CP1251, вывод может быть искажен при чтении через `UTF-8`.
- Парсинг строк в `TestingResultsProcessor` использует пробелы для разделения; при наличии лишних пробелов можно улучшить парсинг, используя `line.trim().split("\\s+")`.
- Защита от деления на ноль в статистике: если в проекте возможен запуск без абитуриентов, можно добавить проверку `total == 0` перед вычислением процентов.

**Запуск проекта**
Собрать и запустить все классы из корня проекта можно командами:

```bash
javac *.java
java Main
```

После запуска введите одну из тем (`1.1`, `3.9`, `4.4`, `5.5`, `6.1`) и следуйте подсказкам меню.
