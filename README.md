**Проект**: Java_Lab_5

- **Краткое описание**: набор небольших учебных задач на Java: работа с дробями, вычисление симметрической разности списков, обработка результатов тестирования абитуриентов, поиск звонких согласных в тексте и демонстрация разворота очереди через стек.

**Файлы проекта**
- `Main.java`: главный класс с текстовым меню. Запускает разные подсистемы по выбору темы:
  - `1.1` — работа с дробями (`Fraction`, `FractionDemo`)
  - `3.9` — формирование списка L (симметрическая разность) (`ListSymmetricDifference`)
  - `4.4` — обработка результатов тестирования (`TestingResultsProcessor`, `Applicant`, `TestingDemo`)
  - `5.5` — поиск звонких согласных в тексте (`VoicedConsonantFinder` / `VoicedConsonantAnalyzer`)
  - `6.1` — демонстрация разворота очереди через стек (`ReverseQueueArrayStack.QueueReverser`)

- `Fraction.java`: класс `Fraction` реализует интерфейс `FractionOperations`. Поддерживает нормализацию знака, проверку знаменателя, кэширование вещественного значения, методы установки/получения и сокращение дроби.

- `FractionDemo.java`: демонстрация использования `Fraction`: создание дробей, кэширование вычислений, сравнение, сокращение, ввод пользователем.

- `ListSymmetricDifference.java`: содержит `SymmetricDifferenceProcessor` — вычисляет симметрическую разность списков L1 и L2 и предоставляет методы для подробного анализа.

- `ReverseQueueArrayStack.java`: содержит вложенный статический класс `QueueReverser`, который принимает очередь `Queue<Integer>` и печатает её элементы в обратном порядке, используя массив и `Stack`.

- `Applicant.java`: модель абитуриента с валидацией входных данных, геттерами и удобными представлениями (`toString`, `toShortString`).

- `TestingResultsProcessor.java`: класс для обработки результатов тестирования: парсинг строк, хранение списка абитуриентов, подсчёт статистики допущенных/недопущенных и вывод статистики.

- `TestingDemo.java`: демонстрация работы обработки тестовых данных. Вызов `Main.printResults(processor)` показывает результаты.

- `VoicedConsonantFinder.java`: содержит `VoicedConsonantAnalyzer` и утилиты `RussianTextUtils` для поиска звонких согласных в тексте. Поддерживается загрузка из файла.

- `file_task5.txt`: пример текстового файла для задачи `5.5` (может использоваться как входной файл). Поместите в корень проекта.
